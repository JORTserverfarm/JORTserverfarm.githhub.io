About Us

We are a student engineering group from Warren Tech Central, passionate about building innovative systems and exploring real-world computing challenges.

This portfolio showcases our Raspberry Pi server cluster project, a modular, high-performance server designed for lightweight computing and experimentation.

The project was developed as part of NASA HUNCH (High School Students United with NASA to Create Hardware), an initiative that connects students with NASA engineers to work on practical, hands-on projects that address real engineering problems.

Our work combines design, data analysis, and hardware implementation to create a complete system from sketches and CAD models to a fully functioning server farm.
